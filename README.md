**Converts sheet music/midi into playable Heartopia "sheets"**

**(Alpha version)**





Workflow:



* Click open file and find your midi/xml file
* Hit convert
* The output is Heartopia formatted notes





What does x dropdown do:



Output format: Instrument key style**(15 or 22 as of now)**

Transpose: Pitch

XML octave correction: use if xml file output says aaa or something like that

Grouping: **(Not Available yet, working on it)** Choose between sheet faithful formatting or easily readable format
